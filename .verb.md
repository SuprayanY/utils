## Usage

To get all utils grouped by collection:

```js
var utils = require('{%= name %}');

// All utils are on the `_` property
console.log(utils._);
```

**Get a specific collection**

```js
var utils = require('{%= name %}');

// get only the string utils
var string = utils.string;
```
